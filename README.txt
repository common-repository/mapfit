Easily add a map to your WordPress site or post. Fill in an address form, choose a map style, and deploy to a page or your post using shortcodes or your page builder.

== Description ==
Mapfit helps you create maps your users will enjoy, and fits your page style. We offer you the features and tools to build complex or simple map displays of your location, region or the world. Display your map on a WordPress page or WordPress post. All of our maps, map markers and interactions are professional designed to provide the best possible user experience. Our map plugin is free for WordPress users. Visit us at www.mapfit.com for more information about how we build our maps.

Our map widget is completely free to use with activation.

= Feature list = 
* Easy to use form accepts an address as input
* Many professionally designed map markers to customize your map location
* Day and Night themes to style the map background (more coming soon)
* Full control of map size and page placement 
* Free walking, driving and biking directions requests (coming soon)

== Installation ==
= Thanks for choosing Mapfit! = 
1. Install the Mapfit Plugin available in WordPress plugin market.
2. Activate the plugin through the 'Plugins' menu in WordPress (email and domain required for activation).
3. Create your Mapfit map and click 'Save your map' when done.
4. Press the 'Connect Mapfit' button to enable the shortcode. 
5. Copy the shortcode and paste it on any page of your WordPress site or WordPress post.

= Adjust the map size =
The default Mapfit map size is width="100%" and height="500px" as seen in the shortcode. Adjust the actual map display size as needed within your page or post.
